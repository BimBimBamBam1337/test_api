## 📋 О проекте

Проект разработан с использованием **Python 3.13** и следует принципам **чистой архитектуры** (Clean Architecture), что обеспечивает:

- **Независимость от фреймворков** - бизнес-логика не зависит от внешних библиотек
- **Тестируемость** - каждый слой можно тестировать изолированно
- **Гибкость** - легкая замена внешних компонентов (БД, внешние сервисы)
- **Поддерживаемость** - четкое разделение ответственности между слоями

## 🚀 Быстрый старт

### Предварительные требования

- Docker и Docker Compose
- Make (для удобства использования команд)

### Запуск проекта

```bash
# Сборка и запуск контейнеров
make up

# Применение миграций базы данных
make migrateup

# Запуск тестов
make tests

# Проект доступен по адресу: http://localhost:8000
```
